object.upload_s3(local_asset, bucket, key_prefix=None, key=None)¶

Uploads a local asset (file or files) to s3

Parameters

  • local_asset (pathlib.PosixPath) – The specifier for the local asset

  • bucket (str) – the AWS S3 bucket name

  • key_prefix (str) – The prefix used for the key name in the bucket

  • key (str) – the key name to use for the remote asset

The key_prefix and key are optional. The key_prefix is used after the bucket name and before the key (as a folder structure). the key is used to have a different name of the assest. If the asset is to be named the same as the input, this can be excluded.
